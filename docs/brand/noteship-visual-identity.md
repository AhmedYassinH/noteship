# Noteship — Visual Identity & UI Personality
_Last updated: 2026-01-24_

## 1) Visual direction (locked)
**Modern editorial (subtle)** for public pages + **calm utility** for the app.

Keywords:
- Calm, minimal, readable, precise
- “Paper + precision”, not “neon AI”

## 2) Color philosophy
- **Neutrals-first** (light backgrounds, soft text)
- One restrained accent color for primary actions
- Avoid loud gradients and neon saturation
- Use color for meaning (status, warnings) not decoration

## 3) Contrast & accessibility
- Prioritize readability and long-session comfort
- Avoid low-contrast gray-on-gray
- Support dark mode (app), optional for marketing

## 4) Shape language
- Subtle rounding (not bubble UI)
- Thin borders, soft shadows (sparingly)

## 5) Iconography
- Simple line icons, consistent stroke weight
- Avoid cartoon icons or complex illustrations
- Icons should support scanning (not become “art”)

## 6) “Quiet power” rules
Do:
- generous whitespace
- clear hierarchy
- consistent spacing tokens

Avoid:
- too many cards
- excessive shadow layers
- decorative blobs
- playful AI mascots

## 7) Screenshot style (marketing)
- Show real product UI (not mock “AI robot” visuals)
- Prefer annotated screenshots (subtle callouts)
- Keep framing consistent (same device frame or none at all)
